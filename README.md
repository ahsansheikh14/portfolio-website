# Ahsan Sheikh — Portfolio Website

Premium personal portfolio built with React, Vite, Tailwind CSS, and Framer Motion.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Before You Deploy

1. **Resume PDF** — Place your CV at `public/resume.pdf` (required for Download CV button).
2. **Site config** — Update links and email in `src/config/site.js`:
   - GitHub username & URLs
   - LinkedIn URL
   - Email address
3. **Project links** — Add GitHub/live URLs in `src/data/projects.js`.

## Build

```bash
npm run build
npm run preview
```

## Deploy

Deploy the `dist` folder to [Vercel](https://vercel.com), Netlify, or GitHub Pages.

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React icons
